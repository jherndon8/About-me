Minesweeper game

Instructions

  • Click a square to dig. If there is a mine under that square, you lose. If a number is displayed, that tells you how many mines are next to it (including diagonals).
  • Note: do not worry about the first move. The mines are set after the first dig so you should always have a nice starting point.
  • Right-click a square to mark it as a mine. You do not have to mark every mine to win; you only need to dig all the non-mines. Right-clicking before you dig your first square is disabled.
  • If there is a number with the correct number of mines marked around it, double-click the number to dig on all the squares that are not marked around it (This has the same functionality as right+left clicking in the original game).
